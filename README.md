# NSFW Image Detection Model

This project builds a **deep learning model for detecting NSFW (Not Safe For Work) images**.  
The implementation, training, and evaluation are done in a Jupyter Notebook.

## Project File
- `NSFW_Detection.ipynb` – Contains data preprocessing, model training, evaluation, and prediction.
