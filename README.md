NSFW Image Detection Model

This project builds a deep learning model for detecting NSFW (Not Safe For Work) images.
The implementation, training, and evaluation are done in a Jupyter Notebook.

Project File

NSFW_Detection.ipynb – contains data preprocessing, model training, evaluation, and prediction.

How to Run

Clone the repository:

git clone https://github.com/sharath-hb03/NSFW-image-detection-model.git
cd NSFW-image-detection-model

Open and run the notebook:

jupyter notebook NSFW_Detection.ipynb
Purpose

This project demonstrates how deep learning can be used for automatic content moderation and unsafe image detection.
